Jak Uzyc Klucza

1. Pobierz wszystkie elementy:

curl https://api.szymixsiorek.workers.dev/v1/data -H "X-API-Key: YOUR_KEY"

2. Dodaj element:

curl -X POST https://api.szymixsiorek.workers.dev/v1/data -H "X-API-Key: YOUR_KEY" -H "Content-Type: application/json" -d "{\"name\":\"Test\"}"

3. Pobierz element:

curl https://api.szymixsiorek.workers.dev/v1/data/100 -H "X-API-Key: YOUR_KEY"

4. Edytuj element:

curl -X PUT https://api.szymixsiorek.workers.dev/v1/data/100 -H "X-API-Key: YOUR_KEY" -H "Content-Type: application/json" -d "{\"name\":\"Nowa nazwa\"}"

5. Usun element:

curl -X DELETE https://api.szymixsiorek.workers.dev/v1/data/100 -H "X-API-Key: YOUR_KEY"
