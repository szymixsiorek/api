szymixsiorek API

Cloudflare Workers - Production API

Wszystkie endpoints wymagaja naglowka: X-API-Key: chujcinamatke

O API

To jest prywatne API dostepne tylko z odpowiednim kluczem autoryzacyjnym. API obsluguje operacje CRUD na danych przechowywanych w Cloudflare KV Storage.

  • Autoryzacja poprzez X-API-Key header
  • Przechowywanie danych w KV Storage
  • Obsluga GET, POST, PUT, DELETE
  • Zwracanie JSON
  • Opcjonalny custom ID przy dodawaniu

Dostepne Endpoints

GET /v1

Glowny endpoint API - zwraca liste dostepnych operacji

GET /v1/data

Pobiera wszystkie elementy

POST /v1/data

Dodaje nowy element (wymagane: name, opcjonalnie: id)

Custom ID: Mozesz podac wlasne ID w polu "id".
Jesli ID jest zajete, API automatycznie przydzieli nearest available ID i zwroci informacje o zmianie.
GET /v1/data/{id}

Pobiera konkretny element

PUT /v1/data/{id}

Edytuje istniejacy element

DELETE /v1/data/{id}

Usuwa element

Struktura Danych

Kazdy element ma nastepujaca strukture:

{ "id": "123", // custom ID lub timestamp "name": "string", // wymagane "description": "string", // opcjonalnie "value": any, // opcjonalnie "created_at": "ISO 8601", "updated_at": "ISO 8601" }

Przyklady Uzycia

Pobierz wszystkie elementy

curl https://api.szymixsiorek.workers.dev/v1/data \ -H "X-API-Key: chujcinamatke"

Dodaj element z custom ID

curl -X POST https://api.szymixsiorek.workers.dev/v1/data \ -H "X-API-Key: chujcinamatke" \ -H "Content-Type: application/json" \ -d '{"name":"Test","id":"100"}' Jesli ID 100 jest zajete: { "status": "ok", "message": "Requested ID (100) is taken. Using ID: 101", "item": {...} }

Dodaj element bez custom ID (auto-generate)

curl -X POST https://api.szymixsiorek.workers.dev/v1/data \ -H "X-API-Key: chujcinamatke" \ -H "Content-Type: application/json" \ -d '{"name":"Test"}'

Edytuj element

curl -X PUT https://api.szymixsiorek.workers.dev/v1/data/100 \ -H "X-API-Key: chujcinamatke" \ -H "Content-Type: application/json" \ -d '{"name":"Nowa nazwa","value":999}'

Usun element

curl -X DELETE https://api.szymixsiorek.workers.dev/v1/data/100 \ -H "X-API-Key: chujcinamatke"
